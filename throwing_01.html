Throwing（投擲）スキル

Throwing（投擲）スキルについて、現時点での仕様をまとめています。
なお、このスキルはStygian Abyssでの新種族ガーゴイル専用のものです。

１．武器の種類

投擲武器は以下の3種類となっています。それぞれ攻撃力や速度は異なりますが、共通の特徴があります。
  • 片手武器
  • 射程に幅がある
  • 弓系武器と同じく、攻撃直前に0.75秒の停止時間が必要。
  • ほかの武器とは異なり、命中判定後0.75秒のリカバリータイムが必要（投擲武器が戻る時間）。

投擲武器3種の射程および様々な補正は以下のとおりです。

射程 = 基本射程 + 3 * ((STR - 装備STR) / (140 - 装備STR))

３．マイナス方向の補正緩和


射程とターゲット距離による命中およびダメージの補正のうち、マイナス方向の補正については、それを緩和することも可能です。

（１）ダメージ修正の緩和

装備者のSTRが高ければ、ダメージのマイナス方向の修正を緩和することができます。

緩和率[％] = 180 * ((STR - 装備STR) / (140 - 装備STR)) -90　（0〜90％）

  • soul glaive（装備STR60）を装備したキャラクターのSTRが90だった場合、緩和率は0％より小さいため、
    射程ぎりぎりからの攻撃によるマイナス方向のダメージ修正を緩和することはできない。
  • soul glaive（装備STR60）を装備したキャラクターのSTRが124だった場合、緩和率は54％となり、
    射程ぎりぎりからの攻撃によるマイナス方向のダメージ修正を54％緩和することができる。
    そのため、ダメージ修正は−50％から−23％となる。

（２）命中率修正の緩和（その１）

装備者のDEXが高ければ、マイナス方向の命中率修正を緩和することができます。

緩和率[％] = 確認中　（0〜90％）

緩和率が発生するDEXの値があること、
DEXが同じでも、武器種類によって緩和率が違っていそうなことを確認しています。

（３）命中率修正の緩和（その２）

投擲武器で近接の敵を攻撃するとき、命中率修正をThrowingスキルで緩和することができます。

緩和率[％] = 0.75 * (Throwingスキル) （0〜90％）

近接攻撃の場合、命中率修正の緩和（その１）と（その２）は加算され、最大90％まで緩和することができます。
また、Throwingスキルが120かつDEXが140以上の場合、命中率のマイナス補正がなくなり、＋5％の命中率補正が かかります。

４．盾装備時のペナルティ

投擲武器は片手武器のため、盾も装備できますが、そのとき、命中および回避プロパティが減少します。
盾スキルが0のとき、他に命中および回避プロパティが付加された状態でなければ、命中および回避プロパティは100減少し、 相手に攻撃を命中させることはできず、相手からの攻撃を確実に受ける状態となります。
盾スキルが120のときでも、命中および回避プロパティが20減少することになります。

命中 = -100 + (parryスキル) * 2 / 3

回避 = -100 + (parryスキル) * 2 / 3
